Quick Start

The shortest path from zero to your first session:

  1. 1

    Install VSCode

    Free download from code.visualstudio.com

  2. 2

    Install Git

    Download from git-scm.com — Mac users: it may already be installed

  3. 3

    Install Claude Code

    Open the VSCode terminal (Ctrl+` on Windows, Cmd+` on Mac) and follow the install instructions at claude.ai/code. You'll need a Claude Pro or Max account.

  4. 4

    Download the DOE Starter Kit

    Clone or download from GitHub

  5. 5

    Run the setup script

    ./setup.sh

  6. 6

    Open your project folder

    cd my-project

  7. 7

    Start Claude Code

    Type claude and press Enter

  8. 8

    Start your first session

    Type /stand-up

Not sure what some of that means? That's fine — the Getting Started page walks through every step with screenshots and explanations.
